Object 2

Creați un șir asociativ (obiect) cu următoarele valori:
let product = { id: 3, title: "T-Shirt", price: 290, inStock: true };
Apoi scrieți o funcție care returnează un șir formatat pentru afișarea unei casete de produs în următorul format:
T-Shirt — 290 RON (în stoc / epuizat)
În funcție de valoarea inStock, în text trebuie să apară
„în stoc” dacă valoarea este true, sau „epuizat” dacă este false.
Sugestie: folosiți operatorul ternar pentru o soluție mai scurtă și elegantă.
