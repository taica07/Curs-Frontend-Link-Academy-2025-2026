Array8 - multidimensional (Microexercitiu)

Scrieți un program în care utilizatorul introduce tipul cardului și numărul de rate. Pe baza acestei combinații trebuie să găsiți procentul comisionului din matricea tarifelor.
Pașii:
Întrebați utilizatorul despre tipul cardului (VISA, Master, Amex) și numărul de rate (1, 3, 6, 12).
Validați inputul: dacă utilizatorul introduce o valoare neacceptată → raportați o eroare și cereți o nouă introducere (sau „aruncați” o excepție).
Găsiți comisionul în matricea bidimensională și afișați rezultatul în forma: Comosion pentru CARD la RATE rate: PROCENT%

Opțiuni permise
const paymentMethods = ["VISA", "Master", "Amex"];
const installments = ["1", "3", "6", "12"];
Matricea comisioanelor (rânduri = carduri, coloane = rate)

[0.8, 1.2, 1.9, 3.2], // VISA
[0.7, 1.1, 1.8, 3.0], // Master
[1.0, 1.5, 2.2, 3.5], // Amex
];
